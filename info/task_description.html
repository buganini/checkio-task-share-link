You must find the HTTP/HTTPS hyperlinks in the robots communications.

Given strings contain URL links and must be retreived. URLs typically start with "http://", "https://", or contain HTTP GET query string and allow Unicode string except for whitespace characters.

Input: A string including the URLs.

Output: A list contains all of the URLs in a class string.

Example: 

    assert checkio(u"blah http://台灣.taipei/ blah") == [u"http://台灣.taipei/"]
    assert checkio(u"""
        PyCON APAC 2014 http://pycon.tw/
        en: https://tw.pycon.org/2014apac/setlang/?language=en&next=
        zh: https://tw.pycon.org/2014apac/setlang/?language=zh&next=
        ja: https://tw.pycon.org/2014apac/setlang/?language=ja&next="""
        ) == [
        u'http://pycon.tw/',
        u'https://tw.pycon.org/2014apac/setlang/?language=en&next=',
        u'https://tw.pycon.org/2014apac/setlang/?language=zh&next=',
        u'https://tw.pycon.org/2014apac/setlang/?language=ja&next='
    ]
